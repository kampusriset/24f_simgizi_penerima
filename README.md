# 24f_simgizi_penerima


Project ini dibuat sebagai kelengkapan tugas tengah semester genap mata kuliah Pemrograman Web. Modul ini berfokus pada pengelolaan data penerima manfaat untuk Sistem Informasi Gizi (SIMGIZI).



## 📁 Folder Architecture

```bash
└── penerima-manfaat/
    ├── components/
    │   └── ui/
    │       ├── form-penerima.php
    │       └── form-sekolah.php
    ├── config/
    │   └── database.php
    ├── controllers/
    │   ├── PenerimaManfaatController.php
    │   └── SekolahController.php
    ├── models/
    │   ├── PenerimaManfaat.php
    │   └── Sekolah.php
    ├── public/
    │   ├── css/
    │   │   └── style.css
    │   └── js/
    │       ├── form-sekolah.js
    │       ├── modal.js
    │       └── script.js
    ├── routes/
    │   └── web.php
    ├── views/
    │   ├── form.create.penerima.php
    │   ├── form.edit.penerima.php
    │   └── penerima.php
    ├── .gitignore
    ├── index.php
    └── README.md
```
